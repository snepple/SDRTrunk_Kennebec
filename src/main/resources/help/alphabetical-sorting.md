## Goal
Sort your aliases alphabetically for easier navigation and management.

## Overview
Alphabetical Sorting allows you to quickly organize your alias lists, making it much simpler to find specific entries in large configurations.

> **Note**
> Sorting does not permanently alter the underlying save file order, it only changes how it is displayed.

## Process
1. Navigate to the Alias tab in the playlist editor.
2. Click the column header for the Alias Name.
3. The list will toggle between A-Z, Z-A, and the original unsorted order.

## Component Map
* **Alias Name Header:** Click to sort alphabetically.
* **Sort Indicator:** A small arrow that shows the current sort direction.

For more information on managing your configurations, see the [Playlist Editor](/help/playlist-editor.md) guide.
