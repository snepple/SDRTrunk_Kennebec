# Advanced & System

This section covers everything beyond decoding a channel: getting the application installed and tuned to your machine, staying informed when something needs attention, and the power-user features that make SDRTrunk Kennebec suitable for running 24/7. You do not need any of these to start listening — reach for them as your setup grows.

## Where to start

- New to the application? Begin with **System Requirements** and **User Preferences** to get it installed and configured for your computer.
- Running unattended? Set up **Notifications**, **System Health Notifications**, and **Inactivity Monitoring** so you are alerted the moment something stops working.
- Chasing performance or audio quality? See **GPU Acceleration**, **Smart Bandwidth**, and **Virtual Audio Cable**.

## In this section

| Topic | What it covers |
| --- | --- |
| **System Requirements** | What your computer needs to run SDRTrunk Kennebec smoothly. |
| **User Preferences** | Application-wide settings for audio, directories, and behavior. |
| **Notifications** | Email and Telegram alerts for errors and important events. |
| **System Health Notifications** | Built-in alerts when tuners or channels stop working. |
| **Inactivity Monitoring** | Get notified when a channel goes unexpectedly quiet. |
| **Application Watchdog** | Automatic recovery if the application itself becomes unresponsive. |
| **Diagnostics** | Toggle detailed troubleshooting logs on the fly. |
| **Managing System Logs** | Find, read, and share log files when asking for help. |
| **Audio Recordings** | Record calls to disk and browse the recordings catalog. |
| **Audio Transcriptions** | Transcribe radio audio using OpenAI Whisper or Google Speech-to-Text. |
| **Gemini AI** | Use AI to auto-tune channels, analyze logs, and audit audio quality. |
| **GPU Acceleration** | Offload signal processing to an OpenCL-capable GPU to free up the CPU. |
| **Smart Bandwidth** | Automatically optimize the sample rate to save CPU. |
| **Ignore Unwanted Talkgroups** | Hide talkgroups that are not in your alias list. |
| **Virtual Audio Cable** | Route specific aliases to a virtual audio cable for other applications. |
| **Natural Scrolling** | Smooth, macOS-style scrolling behavior. |
| **Persistent Layouts** | Remember column widths and sorting between restarts. |
