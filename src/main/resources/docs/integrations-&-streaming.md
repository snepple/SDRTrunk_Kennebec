# Integrations & Streaming

Once SDRTrunk Kennebec is decoding calls, it can send that audio out to other people and platforms — think of it as running your own private internet radio station. It can stream live to public feeds, archive calls in searchable databases, push paging tones to first responders, and publish events to home-automation systems. You can run several of these at the same time without slowing anything down.

## Where can I send audio?

| Destination | Best for |
| --- | --- |
| **Broadcastify** | Sharing local public-safety feeds with the world. |
| **Zello** | A private push-to-talk group for you and your team. |
| **Rdio Scanner / OpenMHz** | Searchable, browsable archives of every call. |
| **ThinLine Radio** | Logging and storing completed calls. |
| **IAmResponding** | Sending paging audio to responders' phones. |
| **MQTT** | Publishing tone-detection events to Home Assistant and other automation. |

## In this section
