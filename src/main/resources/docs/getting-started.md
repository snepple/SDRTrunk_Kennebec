# Getting Started with SDRTrunk Kennebec

Welcome to SDRTrunk Kennebec! Think of this software as a smart radio scanner that can listen to multiple conversations at the exact same time.

We've put together a few simple guides to get you listening quickly. Work through them in order using the sidebar on the left:

* **Introduction** — learn what SDRTrunk Kennebec is and how it works, using simple analogies.
* **Quickstart** — step-by-step instructions to get your first channel playing.
* **Call Flow Logic** — understand how a call moves from the airwaves to your speakers.
* **Audio Playback Bar** — get comfortable with the playback controls at the top of the window.

When you're ready to send your audio online, see **Integrations & Streaming → Overview**.

> **Note:**
> If you ever get stuck, just remember that setting up a scanner is like tuning a TV. Once you find the right channel, the application does the rest!
