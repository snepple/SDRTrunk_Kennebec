# 🚀 Getting Started with SDRTrunk Kennebec

Welcome to SDRTrunk Kennebec! Think of this software as a smart radio scanner that can listen to multiple conversations at the exact same time.

We've put together a few simple guides to get you listening quickly:

* 📖 [Introduction](introduction.md) - Learn what SDRTrunk Kennebec is and how it works using simple analogies.
* ⚡ [Quickstart](quickstart.md) - Step-by-step instructions to get your first channel playing.
* 🌐 [Overview of Streaming](overview.md) - Learn how to send your radio calls to the internet so you can listen anywhere.

> [!NOTE]
> If you ever get stuck, just remember that setting up a scanner is like tuning a TV. Once you find the right channel, the application does the rest!
